Accounting Web

Accounting Web is a web appliction. Use to manage your accounting.

Prerequisites:
- Firebase

=====================================
Implement Accounting Web
=====================================

Firebase:
- accounting application require a Firebase database to store it's data. Make sure to update the file "config.json" file.
- go to firebase console https://console.firebase.google.com and create project, database.
- import json file "types.json" to database on data location "/types".
- copy json in the file "rules.json" to update database rules.
